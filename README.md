# Blue Screen Mod

Клиентский мод для Minecraft 1.21.1 Fabric.

## Функционал

- **E** — включает синий экран смерти (наложение синего слоя на экран)
- **0** — отключает синий экран, возвращает нормальное отображение

## Сборка

### Локально
```bash
./gradlew build
```

### Через GitHub Actions
Просто запушь репозиторий на GitHub — сборка запустится автоматически через Actions.

## Установка

1. Установи Fabric Loader для 1.21.1
2. Положи `.jar` из `build/libs/` (без `sources`) в папку `mods`

## ⚠️ Важно

Клавиша **E** конфликтует с инвентарём. Чтобы работало корректно:
- Зайди в настройки управления Minecraft
- Измени клавишу инвентаря на другую (например, `I`)
- Или измени клавишу мода в коде (`GLFW.GLFW_KEY_E` на другую)
